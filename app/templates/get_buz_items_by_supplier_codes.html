Get Buz Items by Supplier Code

This tool allows you to retrieve specific items from a Buz Excel file based on their supplier codes. Simply upload the Excel file containing the Buz items and provide a list of supplier codes (one code per line). The tool will filter the items and generate a new file containing only the rows that match the supplied codes.

This can be useful for isolating items related to specific suppliers or for further analysis of a targeted subset of your inventory.
